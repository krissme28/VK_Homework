# Декомпозиция MVP Магазина приложений (Android)

## Epic 1: Управление данными и списком приложений
**Общая оценка:** 10 дней (80 рабочих часов)

*   **Task 1.1: Реализация кэширования в Room.** Создание локальной БД для хранения списка приложений, чтобы обеспечить работу Offline-first. *(8ч)*
*   **Task 1.2: Обработка состояний загрузки (LCE).** Реализация состояний Loading, Content, Error с использованием StateFlow для главного экрана. *(6ч)*
*   **Task 1.3: Пагинация списка (Paging 3).** Оптимизация отображения большого количества приложений через постраничную загрузку. *(8ч)*
*   **Task 1.4: Валидация данных и Error Handling.** Реализация централизованной обработки сетевых и системных ошибок (No Internet, Server Error) с показом Snackbar. *(8ч)*
*   **Task 1.5: Интеграция поиска и фильтров.** Реализация логики поиска по названию и фильтрации по категориям в реальном времени. *(5ч)*
*   **Task 1.6: Доработка функционала категорий.** Реализация логики разбиения приложений по группам и переключения между ними. *(5ч)*

## Epic 2: Детализация и интерактив карточки приложения
**Общая оценка:** 7 дней (56 рабочих часов)

*   **Task 2.1: Парсинг манифеста APK.** Реализация извлечения разрешений (permissions) и версии SDK для отображения в карточке. *(8ч)*
*   **Task 2.2: Динамический UI скриншотов.** Реализация горизонтального списка скриншотов с возможностью полноэкранного просмотра. *(8ч)*
*   **Task 2.3: Система рейтинга и отзывов.** Верстка блока отзывов и логика отображения среднего рейтинга приложения. *(6ч)*
*   **Task 2.4: Обработка Deep Links.** Настройка навигации в карточку приложения по внешним ссылкам. *(6ч)*
*   **Task 2.5: UI-тестирование переходов (Espresso).** Автоматизированная проверка корректности открытия карточки из общего списка. *(6ч)*

## Epic 3: Модуль установки и управления пакетами
**Общая оценка:** 9 дней (72 рабочих часа)

*   **Task 3.1: Менеджер загрузок (Download Manager).** Реализация фонового скачивания APK-файлов с отображением прогресса в уведомлении. *(8ч)*
*   **Task 3.2: Обработка Install Permissions.** Реализация логики запроса разрешений на установку из неизвестных источников (Android 8.0+). *(6ч)*
*   **Task 3.3: Интеграция PackageInstaller.** Реализация процесса установки через системный API с обработкой результата (Success/Fail). *(8ч)*
*   **Task 3.4: Модуль удаления приложений.** Реализация вызова системного диалога удаления через Intent.ACTION_DELETE. *(6ч)*
*   **Task 3.5: Синхронизация статусов кнопок.** Логика автоматической смены текста кнопки (Установить / Открыть / Удалить) при изменении состояния пакета в системе. *(8ч)*
*   **Task 3.6: Проверка безопасности (Hash MD5).** Реализация проверки целостности скачанного файла перед установкой. *(4ч)*
